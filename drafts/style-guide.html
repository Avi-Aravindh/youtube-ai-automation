Usage Guidelines

📐

Logo Clear Space

Maintain minimum clear space equal to the height of the logo mark on all sides. Never place text or other elements within this zone.

📱

Minimum Size

Logo should never appear smaller than 40px in width for digital use. At profile picture size (98x98px), use the simplified mark version.

🎨

Color Usage

Primary brand color should be used for key elements. Background should always be dark (#0F172A or similar) for optimal contrast and readability.

✅

Thumbnail Best Practices

Use bold, high-contrast text. Keep main message to 3-5 words max. Include faces when possible. Test at mobile size (360px wide) before finalizing.
